**Security Policy**

**Supported versions**
Security fixes target the main branch until the next tagged release.

**Reporting a vulnerability**
Email support@budventure.technology
with:
- Description and impact
- Reproduction steps / PoC
- Relevant logs/version info
Please do not open public issues.

**Our process**
- Acknowledge within 48 hours
- Investigate and propose a fix/mitigation
- Coordinate a responsible disclosure window
- Credit in release notes (optional)
